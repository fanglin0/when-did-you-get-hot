# when-did-you-get-hot

##  What this project is
A visual novel dating sim where you realize your childhood friends suddenly got hot and you try to date them. More of a demo than a completed game so far; 2 endings, completed in 12 hours, click and play. Here's a screenshot :p

## Why I made this project
I was inspried by fratjo and nerdjo but make it less straight. Also title is inspired by Sabrina Carpetner's <a href="https://open.spotify.com/album/1aqg30bNvLSWgShZgX4oop">man's best friend</a>

## How I made this project

Ren'Py. Did the writing and scripting in Ellipsus + VSCode, all assets and backgrounds drawn in Krita.
